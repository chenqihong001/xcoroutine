# CoroutineX - 现代C++协程库

**CoroutineX** 是一个轻量级、高性能的C++20协程库，结合了现代C++的最佳实践，提供直观的异步编程体验。

## ✨ 特性亮点

### 🚀 高性能
- 零动态内存分配的无栈协程
- 编译期任务调度优化
- 最小化协程切换开销

### 🎯 现代化设计
- 纯头文件库（Header-only）
- 基于C++20协程标准
- 概念约束的类型安全接口
- 模块化设计，按需包含

### 🔧 丰富功能
- 多种任务类型：`Task<T>`、`Generator<T>`、`Lazy<T>`
- 灵活的调度器系统
- 协程同步原语：Mutex、Channel、Barrier
- 超时和取消支持
- 堆栈追踪和调试支持

### 📦 易集成
- 无外部依赖
- CMake友好
- 支持包管理器（vcpkg, conan）
